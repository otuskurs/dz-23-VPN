Tun: это виртуальный интерфейс, который работает с данными, относящимися к интернет-протоколам (например, IPv4 и IPv6). Его проще представить как «туннель» для проходящих через него IP-пакетов. Используется, например, в VPN, когда нужно отправить данные через защищенное соединение.

Tap: это тоже виртуальный интерфейс, но он работает с Ethernet-кадрами — теми данными, которые используются в локальных сетях. Tap позволяет эмулировать поведение физических сетей, и его можно использовать в виртуальных машинах, чтобы они могли «видеть» друг друга так, как если бы они находились в одной локальной сети.
Таким образом, главная разница заключается в том, с какими типами данных они работают. Tun — для IP, Tap — для Ethernet.
